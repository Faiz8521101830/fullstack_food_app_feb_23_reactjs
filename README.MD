# To check the node bersion

```
node --version
```

# To initialize the firebase console

```
firebase init
```

# To start the server

```
npm run serve
```

# To create react project using npm & Yarn

```
npm create-react-app foldere_name
```

```
yarn create react-app folder_name
```

# To start the client

```
yarn start
```
